## Reflection and Analysis

### Why do we use Kali Linux? What is the advantage of Kali Linux compared to other Linux distributions?

Kali Linux is a specialized Linux distro meant for cybersecurity, penetration testing, digital forensics, and security assessments. Its main advantage is that it comes with many of pre-installed security tools, which is why there is no need to manually install and configure them. This makes it an efficient platform for both learning and professional security testing.

### Which features and tools of Kali Linux attracted you the most?

The most useful feature of Kali Linux is the large collection of integrated security tools. I found tools such as Nmap, Wireshark, Burp Suite, and John the Ripper particularly interesting. And for someone that is new to cybersecurity the pre-installed tools are a great help for getting started as a student.
